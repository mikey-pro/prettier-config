## **@mikey-pro/prettier-config**

A preset Prettier configuration for use with the Mikey Pro style guide

## Usage

### Install

```shell
npm i -D @mikey-pro/prettier-config
```

### Configure

Extend to Prettier in `package.json`:

```json
{
  "prettier": "@mikey-pro/prettier-config"
}
```
